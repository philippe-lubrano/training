# training

Mise en place de deux bases front-end avec Vite :

- `react-ts/` : React + TypeScript + Tailwind CSS
- `vue-ts/` : Vue + TypeScript + Tailwind CSS

## Lancer les projets

### React

```bash
cd react-ts
npm install
npm run dev
```

### Vue

```bash
cd vue-ts
npm install
npm run dev
```

## Build

```bash
cd react-ts && npm run build
cd ../vue-ts && npm run build
```
